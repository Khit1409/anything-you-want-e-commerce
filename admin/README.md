# Hệ thống quản lý cửa hàng của seller

I. mục đích của hệ thống
- Dành cho seller quản lý cửa hàng.
- Chỉnh sửa cập nhật... sản phẩm của người bán.
- Chỉnh sửa cập nhật quản lý nhân viên của cửa hàng.
- Quản lý đơn hàng, thông tin kinh tế, thu nhập.
- Quản lý thông tin tài khoản ngân hàng.
II. các chức năng dự định
- CRUD sản phẩm.
- Quản lý đơn hàng.
- Quản lý thông tin cửa hàng.
- Quản lý thông tin tài khoản ngân hàng.
- Quản lý danh mục khách hàng ưa thích.
